# CI Survey Brasil

Fleet Configuration Portal - Portal de Controle de Embarcações e Equipamentos

Portal integrado com Google Sheets para gerenciamento em tempo real da frota de embarcações e equipamentos ROV.

## Funcionalidades

- 📊 Dashboard com visão geral da frota
- 📋 Matrix visual de equipamentos por embarcação
- 🏅 Gerenciamento de certificações com alertas de vencimento
- 🔍 Busca em tempo real
- 📱 Design responsivo

## Como usar

Este portal se conecta automaticamente a uma planilha Google Sheets pública. Qualquer alteração na planilha é refletida no site automaticamente.
